# JavaScript String Methods

এই folder এ আমি JavaScript এর গুরুত্বপূর্ণ string methods শিখছি।

## Methods Covered

- slice()
- concat()
- join()
- includes()

## Method Details

### slice()

string থেকে নির্দিষ্ট অংশ কেটে নেয়

### concat()

একাধিক string জোড়া লাগায়

### join()

array কে string এ রূপান্তর করে

### includes()

string এর ভেতরে কোনো word বা character আছে কিনা চেক করে

## Real-Life Use Cases

- Search feature
- Text preview
- Username & full name generate
- Sentence creation

## Best Practice

- Search করার সময় `toLowerCase()` ব্যবহার করা
- Original string modify না করে method ব্যবহার করা
