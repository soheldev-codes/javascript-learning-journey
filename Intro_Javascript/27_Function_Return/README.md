# Function Return & Return Value in JavaScript

এই folder এ JavaScript function এর return কী এবং return value কিভাবে variable এ set করা হয় তা শেখা হয়েছে।

## What is return?

- return function থেকে value বাইরে পাঠায়
- return হলে function execution শেষ হয়

## Syntax

function name() {
return value;
}

let result = name();

## Examples Covered

- Return number
- Return string
- Return boolean
- Return array
- Return object
- Using return value in calculation

## return vs console.log

- console.log() → শুধু output দেখায়
- return → value পাঠায় এবং variable এ store করা যায়

- Function chaining
