# Sort an Array in JavaScript

এই folder এ JavaScript array sort করার বিভিন্ন problem এবং solution দেয়া হয়েছে।

## Problems Covered

1. Sort numbers (ascending)
2. Sort numbers (descending)
3. Sort strings alphabetically
4. Case-insensitive string sort
5. Sort without using sort() method
6. Sort array of objects

## Important Notes

- sort() method string হিসেবে compare করে
- Number sort করতে compare function দরকার
- sort() original array modify করে

## Compare Function

(a, b) => a - b → ascending  
(b, a) => b - a → descending

## Use Cases

- Product price sorting
- Leaderboard ranking
- Search & filter feature
