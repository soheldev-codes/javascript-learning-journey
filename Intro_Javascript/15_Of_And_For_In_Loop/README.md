# JavaScript for...of & for...in Loop

এই folder এ আমি JavaScript এর for...of এবং for...in loop শিখছি।

## for...of Loop

- Array এবং String এর value নিয়ে কাজ করে
- Clean এবং readable

## for...in Loop

- Object এর key নিয়ে কাজ করে
- Object data access করতে বেশি ব্যবহার হয়

## Topics Covered

- for...of with array
- for...of with string
- for...in with object
- for...in with array index
- Simple problem solving

## Important Note

- Array এর value দরকার হলে → for...of
- Object এর key দরকার হলে → for...in
