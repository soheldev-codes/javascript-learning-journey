# Reversing an Array in JavaScript

এই folder এ JavaScript array reverse করার একাধিক উপায় শেখা হয়েছে।

## Methods Covered

1. reverse() method
2. for loop
3. while loop
4. for...of loop

## Important Notes

- reverse() method original array পরিবর্তন করে
- Loop ব্যবহার করলে original array safe থাকে

## When to Use What?

- reverse() → quick solution
- for / while → interview logic
- for...of → clean & readable code

## Use Cases

- Data order change
- Stack / undo feature
- Algorithm practice
