# Return All Even Numbers of an Array

এই folder এ JavaScript function ব্যবহার করে array থেকে সব even number বের করা হয়েছে।

## Problem

একটি array থেকে শুধু even numbers বের করে নতুন array return করতে হবে।

## Logic Used

Even number check:
number % 2 === 0

## Solutions Covered

1. for loop
2. for...of loop
3. while loop

## Why Return a New Array?

- Original array safe থাকে
- Clean functional approach
- Real-life filtering logic

## Use Cases

- Seat number filtering
- Even ID selection
- Data processing
