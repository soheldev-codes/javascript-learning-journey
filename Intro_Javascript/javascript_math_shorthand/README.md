# JavaScript Mathematical Operation Shorthand

এই folder এ আমি JavaScript এর common mathematical operation এর shorthand শিখছি।

## Topics Covered

1. Addition shorthand (+=)
2. Subtraction shorthand (-=)
3. Multiplication shorthand (\*=)
4. Division shorthand (/=)
5. Modulus shorthand (%=)
6. Increment (++) / Decrement (--)
7. Exponent shorthand (\*\*=)
8. Combined examples

## Why Important

- Code short এবং clean হয়
- Frequent variable updates সহজ হয়
- Professional coding style শেখা যায়
