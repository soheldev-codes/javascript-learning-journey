# Multiple Ways to Get & Set Object Properties

এই folder এ JavaScript object এর property get এবং set করার বিভিন্ন উপায় শেখা হয়েছে।

## GET Methods

1. Dot notation → object.property
2. Bracket notation → object["property"]
3. Dynamic key access → object[key]
4. Destructuring
5. for...in loop

## SET Methods

1. Dot notation assign
2. Bracket notation assign
3. Add new property
4. Dynamic property set
5. Update using loop

## When to Use What?

- Dot notation → simple & fixed property
- Bracket notation → dynamic / variable key
- Destructuring → clean & short code
- Loop → all properties নিয়ে কাজ করতে

## Use Cases

- User profile update
- Form data handling
- API response manipulation
