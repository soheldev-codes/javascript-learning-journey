# Array Traversal using for & while Loop

এই folder এ JavaScript এ array traversal শেখা হয়েছে for loop এবং while loop ব্যবহার করে।

## What is Array Traversal?

Array traversal মানে array এর প্রতিটা element একে একে access করা।

## Methods Used

- for loop
- while loop

## Examples Covered

- Print all elements
- Print index with value
- Calculate sum of elements

## When to Use Which?

- for loop → index control সহজ
- while loop → condition based traversal

## Use Cases

- Data processing
- Search & filter
- Calculations (sum, average)
