# Sum of All Numbers in an Array Using Function

এই folder এ JavaScript function ব্যবহার করে array এর সব সংখ্যার যোগফল বের করা হয়েছে।

## Problem

একটি array এর সব সংখ্যার sum বের করতে হবে function ব্যবহার করে।

## Solutions Covered

1. for loop
2. for...of loop
3. while loop

## Function Logic

- প্রথমে sum = 0
- loop দিয়ে array traverse করা
- প্রতিটা element sum এর সাথে যোগ করা
- শেষে sum return করা

## Use Cases

- Cart total price
- Marks calculation
- Data analytics basic logic
