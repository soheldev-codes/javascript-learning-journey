# Introduction to JavaScript Objects

এই folder এ JavaScript এর Object, Properties এবং Values শেখা হয়েছে।

## What is an Object?

Object হলো key-value pair এর collection।
এটি real-life entity represent করতে ব্যবহৃত হয়।

## Topics Covered

- Object creation
- Accessing properties
- Updating values
- Adding & deleting properties
- Object.keys()
- Object.values()
- for...in loop

## Property & Value

- Property (key) → name, age, country
- Value → "Sohel", 22, "Bangladesh"

## Access Methods

- Dot notation → object.property
- Bracket notation → object["property"]

## Use Cases

- User profile
- Product details
- Settings & configurations
