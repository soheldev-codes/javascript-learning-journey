# JavaScript Numbers Fundamentals

এই folder এ আমি JavaScript Numbers এর basic concept শিখছি।

## Covered Topics

1. Number type (integer, decimal)
2. Arithmetic operations (+, -, \*, /, %, \*\*)
3. Special Number values (Infinity, -Infinity, NaN)
4. Math object (round, floor, ceil, sqrt, random, etc.)
5. Type conversion (Number ↔ String)
6. Number methods (toFixed, toString, isInteger)

## Why Important

- JavaScript এ calculation করতে Number জানা must
- Real-life application: price, marks, age, result calculation
- Foundation for advanced JS topics
