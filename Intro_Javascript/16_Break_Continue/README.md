# JavaScript break & continue

এই folder এ আমি JavaScript loop এর break এবং continue keyword শিখছি।

## break

- Loop পুরোপুরি বন্ধ করে দেয়
- Specific condition এ loop stop করতে ব্যবহার হয়

## continue

- Current iteration skip করে
- Loop চালু থাকে

## Topics Covered

- break in for loop
- continue in for loop
- break with array
- continue with array

## Key Difference

- break → loop বন্ধ
- continue → iteration skip
