# JavaScript String Comparison

এই folder এ আমি JavaScript string comparison শিখছি।

## Methods Used

- toLowerCase()
- toUpperCase()
- trim()

## Why Important

- User input clean করতে
- Case-sensitive problem avoid করতে
- Login, form, validation এ ব্যবহার হয়

## Best Practice

- Compare করার আগে string clean করা
- trim() + toLowerCase() ব্যবহার করা

## Topics Covered

- Lowercase conversion
- Uppercase conversion
- Remove extra spaces
- Real-life comparison examples
