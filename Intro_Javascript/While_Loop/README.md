# JavaScript While Loop

এই folder এ আমি JavaScript এর while loop শিখছি।

## What is While Loop?

- Condition true থাকলে loop চলতে থাকে
- Condition false হলে loop বন্ধ হয়

## Topics Covered

- Basic while loop
- Increment & decrement
- Even number print
- Reverse counting
- Sum calculation
- Multiplication table

## Important Note

- Loop এর ভিতরে update না করলে infinite loop হতে পারে
