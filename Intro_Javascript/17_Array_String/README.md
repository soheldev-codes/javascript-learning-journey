# JavaScript String & String vs Array

এই folder এ আমি JavaScript এর String এবং String vs Array শিখছি।

## String

- Text data store করে
- Immutable (direct change করা যায় না)
- Index ও length থাকে

## Array

- Multiple value store করে
- Mutable (change করা যায়)
- Index ও length থাকে

## Key Differences

- String → text
- Array → list of values
- String change করা যায় না
- Array change করা যায়
