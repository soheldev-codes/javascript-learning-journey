# Reverse a String in JavaScript

এই folder এ JavaScript ব্যবহার করে string reverse করার ৪টা ভিন্ন উপায় শেখা হয়েছে।

## Methods Covered

1. split + reverse + join
2. for loop
3. while loop
4. for...of loop

## for...of Loop Explanation

for...of loop string এর প্রতিটা character নিয়ে কাজ করে।
Reverse করার সময় character টাকে শুরুতে যোগ করা হয়।

Example:
reversed = char + reversed
